# PizzaGo

Final project

Web application for ordering a delivery of <pizza>

# Technology stack:

Java

Spring Boot, Spring Security, Spring Data Jpa

Thymeleaf

PostgreSQL

EmailSender (you can reset your password if you forgot it)
